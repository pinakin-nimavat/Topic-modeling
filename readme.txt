How to run the code?

- Install the requirements file: pip install -r ../requirements.txt

- Install en_core_sci_md using: pip install ../en_core_sci_md-0.5.0.tar.gz ( download the package and use the local path)

Restart runtime and run import statements. (If you are using collar cells to install packages)

- set the path for dataset in the file_path variable. (After import statements)
 
Run the notebook either in google colab or Jupyter notebook.
